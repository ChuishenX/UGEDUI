# UGEDUI(有个堆)简介

UGEDUI: Yo**U**-**G**et **E**ngine **D**ownloader with **U**ser **I**nterface

简而言之，就是主要使用You-Get万能下载器进行下载的懒人/新手便于上手的下载器。

尽管听上去如此简单，我们仍然进行了部分开发与其他模块融合，使下载器支持一些You-Get原先不支持的下载(例如网易云音乐的歌单)。

为了更大的便利用户，我们会开发搜索功能
